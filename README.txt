========================================
服飾分類系統 - Qwen3 VL 8B
========================================

這是一個使用 Qwen3-VL-8B-Instruct 模型的服飾圖片分類系統
可以識別洋裝、裙子、褲子、短褲、上衣、外套、褲襪等多種服飾類別

========================================
系統需求
========================================

1. Python 3.8 或更新版本
2. NVIDIA GPU（建議至少 16GB VRAM）
3. 至少 40GB 可用磁碟空間
4. 穩定的網路連線（首次下載模型時需要）

========================================
安裝步驟
========================================

請依序執行以下批次檔：

步驟 1: 安裝 Python 套件
   執行: 1_install.bat
   說明: 安裝所有必要的 Python 套件，包含 PyTorch、Transformers、Gradio 等

步驟 2: 下載 Qwen3 VL 8B 模型
   執行: 2_download_model.bat
   說明: 下載 Qwen3-VL-8B-Instruct 模型（約 16GB）
         模型會儲存在 ./qwen3_vl_model 資料夾
         只需執行一次，之後會自動使用快取的模型

步驟 3: 啟動應用程式
   執行: 3_run_app.bat
   說明: 啟動 Gradio 網頁界面
         程式會在 http://localhost:7860 執行
         瀏覽器會自動開啟界面

========================================
使用方法
========================================

1. 啟動程式後，會開啟網頁界面
2. 點擊「載入模型」按鈕，等待模型載入完成
3. 上傳一張或多張服飾圖片
4. 點擊「開始分類」按鈕
5. 查看分類結果

========================================

========================================
常見問題
========================================

Q: 程式無法啟動？
A: 請確認已依序執行 1_install.bat 和 2_download_model.bat

Q: GPU 記憶體不足？
A: Qwen3-VL-8B 需要較大的 GPU 記憶體
   如果 GPU 記憶體不足，可以嘗試：
   - 關閉其他佔用 GPU 的程式
   - 修改 app.py 中的 torch_dtype 為 torch.float8（需要支援）

Q: 模型下載失敗？
A: 請檢查網路連線，確保能訪問 Hugging Face
   如果在中國大陸，可能需要設定鏡像站

Q: 分類結果不準確？
A: Qwen3-VL-8B 是通用視覺語言模型
   如需更高準確度，建議使用標註資料進行微調
